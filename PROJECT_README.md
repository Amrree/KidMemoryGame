## 🚀 Quick Start Guide

### For Flutter Development
```bash
cd kid-memory-app/
flutter pub get
flutter run
```

### For Art Generation
```bash
# Choose your tool:
cd art-generator-tools/cursor-art-generator/    # For Cursor IDE
cd art-generator-tools/full-branding-generator/  # For complete branding
cd art-generator-tools/cli-tool/                 # For CLI automation
```

```bash

# Modify themes, music, and screenshots
```

### For Android Development
